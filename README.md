# Google-Suite-Quick-Open
Abrir .doc/.docx, .xls/.xlsx e .ppt/.pptx no navegador direto no Google Docs, Sheets ou Slides com duplo clique no Windows.
Google Suite Quick Open: abre automaticamente arquivos Word, Excel e PowerPoint no Google Docs/Sheets/Slides ao dar duplo clique, sem precisar instalar o Office.
Windows shell launcher que intercepta arquivos de Office/Office-like e abre no Google Docs/Sheets/Slides via Google Drive (upload temporário + sessão de edição no navegador).
